# Verify on WordPress

## Step 1: Copy your meta tag

Find your verification meta tag on your [account page](https://clearwatch.app/manage).

## Step 2: Install a header code plugin

1. Go to your WordPress admin
2. Navigate to **Plugins → Add New**
3. Search for **"WPCode"** (formerly Insert Headers and Footers)
4. Install and activate the plugin

## Step 3: Add the meta tag

1. Go to **Code Snippets → Header & Footer**
2. Paste your meta tag in the **Header** section
3. Click **Save Changes**

## Step 4: Verify

Go back to your [Clearwatch account page](https://clearwatch.app/manage) and click **Verify** on your site.

<Note>If you prefer not to install a plugin, you can also add the meta tag directly to your theme's `header.php` file under Appearance → Theme File Editor.</Note>
